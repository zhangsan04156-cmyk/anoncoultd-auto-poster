# ANON 自動發文與排程工具 (ANON Poster)

這是一個基於 Python 與 Playwright 開發的自動化網頁操作腳本，專門用於在 ANON 平台上進行定時、多則訊息的自動化投稿與排程測試。

## ✨ 核心功能

* **無頭模式執行 (Headless Mode)：** 瀏覽器完全在背景默默執行，不干擾日常電腦操作。
* **動態遞迴輸入：** 友善的終端機互動介面，支援連續輸入多則不同的投稿內容。
* **排程防呆機制：** * 自動判斷投稿數量。若只有一則訊息，則不觸發等待時間。
  * 支援自訂每則訊息的間隔時間（分鐘），避免短時間內大量發文造成伺服器異常負荷。
  * 自動判斷最後一則訊息，發送完畢後立即結束程式，不浪費等待時間。

## 🛠️ 技術棧 (Tech Stack)

* Python 3
* Playwright (Chromium)

## release檔功能限制

目前release檔只支援海山的，且固定為無頭模式，
